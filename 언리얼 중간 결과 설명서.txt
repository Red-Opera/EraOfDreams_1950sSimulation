두 번째 작품 : 언리얼 중간 과정

개요 
	- Unreal 시뮬레이션 게임으로 1950년대 미국 분위기를 체험할 수 있는 게임
	- 현재 만든 부분은 시뮬레이션 중 리듬 게임으로 만들 부분으로 사회자와 연주자가 있고 플레이어는 키보드로 연주하여 연주자를 도와주는 게임
	- 제작 기간 1년 계획 중 3주 동안 만든 작품
	- Concert 맵에서 실행

구현 및 최적화 내용
	- 사회자 애니메이션 구현
	- 콘서트 분위기 제작
	- 콘서트 커튼 에니메이션 구현
	- 콘서트 커튼 클로딩 기능으로 실제 커튼 느낌으로 구현
	- 콘서트 관객 애니메이션 추가
	- 플레이어 이동 애니메이션 제작
	- 플레이어 이동 및 점프 구현
	- 플레이어 가만히 있을 시 마우스로 플레이어 관찰 기능
	- 가져온 복잡한 Mesh들을 플레이어가 눈치채지 못할 정도의 간단하게 Mesh 수 감소
	- 임시 배경 음악 추가
	- 시뮬레이션의 미니 리듬 게임인 콘서트 엔딩 느낌 구현

추가 구현 (버그 수정, 편리성 포함) 및 최적화할 내용 [기말에 추가]
 	- 구현
		- 연주자가 연주할 음악 추가
		- 음악 연주시 연주하는 제목 왼쪽 아래에 작게 출력하는 UI 제작
		- 연주할 악보 UI로 띄우기
		- 악보 UI에 들어갈 음표 제작
		- 입력에 따라 연주되는 기능 및 음표 추가 기능 제작
		- 정확, 미스 판정 제작
		- 정확, 미스에 따라 음표 색깔 변경 기능
		- 플레이어 시뮬레이션 퀘스트 UI 구현
		- 플레이어 점프 애니메이션 추가
		- 다음 맵으로 이동하는 텔레포트 기능
		- 콘서트 밖으로 이동하는 기능
		- 1950년대 영상 분위기로 변경 (화면 회색 + 화면 노이즈, 소리 노이즈)

		- 배경 음악 및 효과음 추가
		- 기타 오류 및 버그 제거

	- 최적화
		- 오클루전 컬링으로 플레이어 시야에 보일 경우에만 렌더링
		- LOD 처리 빠른 성능 처리하도록 변경
		- 서브로 존재하는 카메라는 랜더링 회수 감소 (ex : 기존 120fps -> 20fps로 감소)

조작 키
	W : 앞으로 이동
	A : 오른쪽으로 이동
	S : 뒤쪽으로 이동
	D : 왼쪽으로 이동

	왼쪽 마우스 : 공격
	마우스 이동 : 플레이어 카메라 회전
